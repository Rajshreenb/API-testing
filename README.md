TRELLO API TESTING USING POSTMAN:		

Project overview:
    This sample project demonstrates API testing of TRELLO REST ASPIs using postman. The test collection validates the functionlaity of various endpoints by verifying request execution, response data and HTTP status codes.

**TEST COVERAGE:**

 the following validations were performed:
 
 HTTP status code verification:
 
      200- get all boards, get a single board
	    201- create todo list, create donelist and create card
	    400-bad request when something went wrong ot when error comes in picture.
Request and response body validation

API Operations tested:
  Get all boards
	create board
	get single board
	create todo list
	create donelist
	create card
	signup for trello
	move to donelist
	delete board

API endpoints: https://developer.atlassian.com/cloud/trello/rest/api-group-actions/
Outcome:
This sample project demonstrates practical knowledge of REST API testing, request-response validation, and assertions in postman.
